# Alien Invasion

This is a simple 2D game when you play with space ship and shoot aliens fleet.

To run the game you need a Python 3  installed in your system (it does not depend on the OS).

How to run the game:
* Set up virtual environment:
    ```bash
    python -m venv venv
    ```
* Activate virtual environment (example for Mac OS or Linux):
    ```bash
    source venv/bin/activate
    ```
* Install packages from the [requirements.txt](requirements.txt):
    ```bash
    pip install -r requirements.txt
    ```
* Run the game:
    ```bash
    python alien_invasion.py
    ```
